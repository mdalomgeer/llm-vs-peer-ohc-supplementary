# Supplementary Materials

Supplementary materials for the AMIA paper "Comparing Human and Large Language Model Responses to Patients' Online Questions: Towards Multi-dimensional Patient-centered Support".

## Contents

- Prompts. The original user-posted questions submitted to the models verbatim.
- Codebook. The qualitative coding framework, with operational definitions and anchors for each scored dimension and the inter-rater reliability value.
- Prompt-variation example. One worked example showing two wordings of the same case and the divergent model responses, illustrating the prompt sensitivity reported in the paper.

## Method note

Each question was submitted to the models verbatim, with no prompt engineering or follow-up turns, to reflect how a layperson queries these tools rather than an optimized setting. Prompts were fixed to the original user-posted questions for consistency and a fair comparison across models and against peer responses on identical inputs.
