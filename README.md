# Supplementary Materials

Supplementary materials for the AMIA paper "Comparing Human and Large Language Model Responses to Patients' Online Questions: Towards Multi-dimensional Patient-centered Support".

## Contents

- Codebook. The coding framework with the categories and anchors used to score the responses.
- Prompt-variation example. One worked example showing two wordings of the same case and the divergent model responses, illustrating the prompt sensitivity reported in the paper.

## Method note

Each question was submitted to the models verbatim, with no prompt engineering or follow-up turns, to reflect how a layperson queries these tools rather than an optimized setting. Prompts were fixed to the original user-posted questions for consistency and a fair comparison across models and against peer responses on the same inputs.

## Data use

The full set of patient questions and the complete model outputs are not redistributed, to respect the source platform's terms and the study's data-use conditions. The codebook and the worked example let others see the coding scheme and reproduce the prompt setup.
